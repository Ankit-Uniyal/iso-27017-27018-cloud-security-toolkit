# Certification Readiness Checklist - ISO 27017 / ISO 27018

Document ID: CLOUD-CRC-001 | Version: 1.0 | Classification: Internal

A 40-point pre-certification checklist for combined ISO 27001 + ISO 27017 + ISO 27018 Stage 1 / Stage 2 readiness.

## Documentation
- [ ] Cloud Security Policy approved by top management
- [ ] Cloud PII Policy approved (where ISO 27018 is in scope)
- [ ] Cloud Scope Statement signed
- [ ] Cloud RACI matrix in place
- [ ] Cloud Risk Register and Risk Treatment Plan
- [ ] Statement of Applicability (SoA) covering ISO 27017 + ISO 27018 controls
- [ ] Shared Responsibility Matrix per cloud service in scope
- [ ] Cloud Service Agreements and DPAs current

## Implementation Evidence
- [ ] Encryption-in-transit and at-rest evidenced
- [ ] MFA / federation enforced for cloud admins
- [ ] CSPM / configuration baseline tooling in place
- [ ] VM and container hardening standards applied
- [ ] Tenant segregation tested (CSP role)
- [ ] Cloud monitoring with tenant access (CLD.12.4.5)
- [ ] Backup and DR tested with documented results
- [ ] Sub-processor list current and DPAs flow-down (ISO 27018 A.4)

## Privacy Controls (ISO 27018)
- [ ] Documented processing instructions captured (A.1)
- [ ] Marketing exclusion control evidenced (A.2)
- [ ] Sub-processor disclosure mechanism live (A.3)
- [ ] Geographic transparency disclosure to CSC (A.5)
- [ ] DSR cooperation runbook tested (A.9)
- [ ] Breach notification SLA validated through tabletop (A.8)
- [ ] Return / disposal procedure with deletion certificate (A.18)
- [ ] Law enforcement disclosure procedure (A.22)

## Performance and Audit
- [ ] Cloud KPI dashboard live and reviewed monthly
- [ ] Cloud awareness training completion >= 95%
- [ ] Internal audit covering cloud controls completed in last 12 months
- [ ] Management review held with cloud agenda items
- [ ] Nonconformities tracked with root cause and corrective actions

## Operational Readiness
- [ ] Independent assurance reports current (SOC 2 / ISO 27001 / 27017 / 27018)
- [ ] Sub-processor due diligence files complete
- [ ] Customer trust portal updated
- [ ] Public privacy notice / Cookie notice aligned
