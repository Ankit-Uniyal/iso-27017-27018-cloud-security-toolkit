# Cloud Service Agreement (CSA) Review Checklist

Document ID: CLOUD-CSA-001 | Version: 1.0 | Classification: Internal

Use this checklist before signing or renewing any Cloud Service Agreement (CSA) or Data Processing Agreement (DPA) to confirm coverage of ISO 27017 and ISO 27018 obligations.

## Service and Scope
- [ ] Service description, type (IaaS / PaaS / SaaS), and tenancy model documented
- [ ] Geographic regions for storage and processing disclosed
- [ ] Sub-processor list and notification mechanism in place
- [ ] CSP certifications attached (ISO 27001, 27017, 27018, SOC 2)

## Security
- [ ] Shared Responsibility Matrix attached and signed
- [ ] Encryption-in-transit and at-rest specified (algorithms, KMS, BYOK options)
- [ ] Identity / MFA / federation requirements covered
- [ ] Tenant segregation, VM and container hardening described (CLD.9.5.1, CLD.9.5.2)
- [ ] Cloud monitoring / log access provided to CSC (CLD.12.4.5)
- [ ] Vulnerability management and patching SLAs defined

## Privacy (where PII is processed)
- [ ] Documented processing instructions captured (ISO 27018 A.1)
- [ ] Purpose limitation; no marketing / advertising without explicit consent (A.2)
- [ ] Sub-processor disclosure and flow-down (A.3, A.4)
- [ ] DSR cooperation, response timelines (A.9)
- [ ] Breach notification SLA to CSC (A.8)
- [ ] Cross-border transfer mechanism (SCCs / BCRs / TIA) (A.5, A.15)
- [ ] Law enforcement disclosure handling (A.22)
- [ ] Return / disposal terms with deletion certificate (CLD.8.1.5, A.18)

## Operational
- [ ] Service levels (availability, RTO/RPO, support response) defined
- [ ] Incident communication channels and escalation
- [ ] Audit and assurance rights
- [ ] Exit assistance and data portability
- [ ] Change-management notice obligations

## Commercial and Legal
- [ ] Liability and indemnities for security and privacy events
- [ ] Insurance requirements (cyber, professional indemnity)
- [ ] Confidentiality and IP terms
- [ ] Governing law and dispute resolution
- [ ] Notice and termination terms
