# iso-27017-27018-cloud-security-toolkit
A practical implementation toolkit for ISO/IEC 27017:2015 (Cloud Security Controls) and ISO/IEC 27018:2019 (PII Protection in Public Clouds) — gap assessment, cloud risk register, SoA with all extended/cloud-specific controls, CSP/CSC shared-responsibility templates, and Python GRC automation scripts aligned to ISO 27001:2022 and ISO 27002:2022.
